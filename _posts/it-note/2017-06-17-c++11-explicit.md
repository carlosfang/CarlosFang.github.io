用来声明类构造函数是显示调用的，而非隐式调用，**可以阻止调用构造函数时进行隐式转换**。

```c++
#include <iostream>
#include <cstring>
using namespace std;

class A
{
public:
    int var;
    explicit A(int tmp)
    {
        var = tmp;
        cout << var << endl;
    }
};
int main()
{
    A ex(100);
    A ex1 = 10; // error: conversion from 'int' to non-scalar type 'A' requested
    return 0;
}
```

说明，发现隐性调用： 

```c++ 
A ex1 = 10;
A ex2(10); 
A ex1 = ex2; 
```
