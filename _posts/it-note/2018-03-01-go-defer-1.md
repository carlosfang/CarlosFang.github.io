请写出输出结果

```go
func main(){
    a := 1
    b := 2
    defer calc(a, calc(a,b))
    a = 0
    defer calc(a, calc(a,b))
}
func calc(x,y int){
    fmt.Println(x,y,x+y)
    return x+y
}

```

结果是

```sh
1 2 3
0 2 2
0 2 2
1 3 4
```

#### 考察两个知识点：

1.defer是栈调用，后写的先执行

2.defer的函数调用语句会在父函数调用后执行，但是用到的参数会在当时就执行得出
