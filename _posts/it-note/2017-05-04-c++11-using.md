# 一、继承构造函数

用 using 可以把**基类中的构造函数悉数继承到派生类**

```c++
struct A {
    A( int i) {} 
    A( double d, int i) {} 
    A( float f, int i, const char* c) {} // ... 
}; 

struct B : A { 
    using A::A; // 继承 构造 函数 
    // ... 
    virtual void ExtraInterface(){}
 };
```
